welcome to test websocket server, and it would help you blow.
1. configure you web server, and set the 'www' to website root
2. start up websocket server by 'php webserver.php host port'
3. run 'tests/server/init.sh' to set up user and group data
4. run init.php by 'php init.php' to create websocket html client files, and it was named by 'client0.html', 'client1.html' ...
5. open your browser to visit 'http://xxx.com/client0.html'
may you success !
